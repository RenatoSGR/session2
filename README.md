# session2

This repository contains files for session 2.

## Contents

- `test.txt` - A test file used in this session.
- `calculator.py` - Simple calculator module with basic arithmetic operations.
- `test_calculator.py` - Unit tests for the calculator module.

## Running Unit Tests

Install the required dependency and run the tests with:

```bash
pip install pytest
pytest test_calculator.py -v
```
